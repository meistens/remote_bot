Remote Jobs Telegram Bot

Bot is running and ready to receive webhooks!

How to Use:

  1. Click here to open the bot on Telegram or search for @AllJobsGlobalBot
  2. Send /start to begin
  3. Use /jobs to search for remote jobs
  4. Use /help for more commands

Available Commands:

  • /jobs - Get latest remote jobs (default: 5 jobs)
  • /jobs --count 10 - Get specific number of jobs (max 50)
  • /jobs --geo USA - Filter by location
  • /jobs --industry tech - Filter by industry
  • /jobs --tag python - Filter by technology
  • /help - Show help message

Example Usage:

/jobs --count 3 --geo USA --tag golang
Get 3 Golang jobs in the USA
/jobs --industry tech --tag python
Get Python jobs in tech industry
Webhook Endpoint: POST /api/webhook
